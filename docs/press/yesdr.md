# YESDR Standard

YESDR (Your Extensible Software Defined Radio) is a modular digital radio standard framework developed by Chandhar Research Labs Pvt. Ltd. and published by
Basig Wireless Press.

## Scope
- Extensible SDR systems
- Modular PHY / MAC abstractions
- Research‑driven protocol design
- Academic and experimental deployments

YESDR serves as a reference standard for education,
research, and system prototyping.

Official documentation and specifications for the YESDR standard framework.  
🔗 [https://yesdr-standard.org](https://yesdr-standard.org)
